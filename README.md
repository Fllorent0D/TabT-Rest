# Backend of BePing 2

Transform call SOAP Tabt API via REST
Caching server for requests

## Running Locally with caching enable

```sh
$ cd beping2-backend
$ npm install
$ npm start
```

## Running Locally with caching disable

```sh
$ cd beping2-backend
$ npm install
$ npm start
```
